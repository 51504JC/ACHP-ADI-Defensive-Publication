# Abstract

## Axiom Conflict Handling Protocol with Action Dispatch Interface (ACHP-ADI): Deterministic Handling and Attestation of Unresolved Decision States in Probabilistic AI Systems

This disclosure describes a control architecture for computing systems that process probabilistic inference outputs. The architecture comprises an independent authority-state derivation layer that evaluates a probability distribution from an inference engine against fixed thresholds to determine whether the decision margin between top-ranked candidates is sufficient for action authorization. When the margin is insufficient, the system enters a defined conflict condition in which no decision object is ever instantiated — including in transient, register-level, speculative, or ephemeral machine states. Instead of a decision, the system generates a negative attestation: a cryptographically signed artifact documenting that no decision could be produced, containing the full distribution and decision metadata sufficient for deterministic replay of the decision boundary. The artifact is appended to a tamper-evident, hash-chained log. All downstream action dispatch passes through a non-bypassable execution interface that operates as a state-driven emitter, has no data path from inference outputs to action emission, and is structurally absent the computational primitives necessary to derive actions from inference data. This structural absence is enforced by hardware mechanisms: either the execution interface lacks arithmetic processing units, or inference-domain data is excluded from its address space by hardware-enforced memory isolation. The architecture converts probabilistic uncertainty into a verifiable, non-overridable control state, enabling independent verification that ambiguous inferences never caused uncontrolled action execution.

**Keywords:** non-instantiation, AI safety, execution gating, negative attestation, deterministic replay, probabilistic inference, edge computing, cryptographic audit, decision margin, safety-critical systems
